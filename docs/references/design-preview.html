Introducción a las Variables

Las variables son contenedores que almacenan valores. En JavaScript moderno, usamos let y const para declararlas.

"Una variable bien nombrada vale más que mil comentarios."

Reglas de nombrado

  • > Deben comenzar con letra, _ o $
  • > Son case-sensitive (name ≠ Name)
  • > No pueden ser palabras reservadas

Para más detalles, consulta la documentación oficial de MDN.
