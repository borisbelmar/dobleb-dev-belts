# Variables y Tipos de Datos

Las variables son contenedores que almacenan valores.

## Declaración

En JavaScript moderno usamos `let` y `const`:

```js
const name = "devBelts";
let level = "white";
```

## Tipos primitivos

- `string` — texto
- `number` — números
- `boolean` — verdadero/falso
- `null` — ausencia intencional de valor
- `undefined` — valor no asignado

## Desafíos

1. Declara una variable para cada tipo primitivo
2. Intenta reasignar una variable `const` y observa el error
3. Usa `typeof` para verificar el tipo de cada variable
