El sistema operativo (Linux, Windows, macOS, Android, iOS) es la capa de software entre el hardware y las aplicaciones. Sin él, cada programa tendría que comunicarse directamente con cada componente físico de la computadora.

## ¿Qué hace?

- **Gestiona recursos**: decide qué programa usa la CPU y cuánta memoria tiene cada uno
- **Abstrae el hardware**: las aplicaciones no necesitan saber cómo funciona el disco duro
- **Maneja archivos**: organiza, lee y escribe datos en el almacenamiento
- **Proporciona interfaz**: gráfica (ventanas) o de texto (terminal)
