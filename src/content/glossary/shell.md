El **shell** es el intérprete de comandos que conecta al usuario con el sistema operativo. Recibe lo que escribes en la terminal, lo interpreta y ejecuta los programas correspondientes.

Los shells más comunes son:
- **bash** (Bourne Again Shell): el más popular, creado en 1989 por Brian Fox para GNU
- **zsh** (Z Shell): más moderno, con autocompletado mejorado y plugins, default en macOS desde 2019
- **PowerShell**: el shell de Windows, orientado a objetos en vez de texto
- **fish** (Friendly Interactive Shell): diseñado para ser amigable con el usuario

El shell se configura mediante archivos como `~/.bashrc`, `~/.zshrc` o `~/.bash_profile`, donde puedes definir alias, variables de entorno y funciones personalizadas.
