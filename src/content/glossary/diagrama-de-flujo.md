Un **diagrama de flujo** usa símbolos estandarizados para representar visualmente un proceso:

- **Óvalo:** inicio o fin del proceso
- **Rectángulo:** acción o proceso
- **Rombo:** decisión (sí/no, verdadero/falso)
- **Paralelogramo:** entrada o salida de datos
- **Flechas:** dirección del flujo

Se usan para documentar algoritmos, procesos de negocio, flujos de autenticación y cualquier lógica con múltiples caminos. Herramientas comunes: Mermaid (diagramas como código), Draw.io, Excalidraw.
